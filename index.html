Mastering CSS.

A complete reference for web designers and developers. Quick, easy, and effective.

01 Selectors

Target and style elements with precision. Think of them as the address you use to mail your styles.

/* Tag Selector */
h1 { color: #f76b90; }

/* Class Selector */
.card { background: #1a1c23; }

/* ID Selector */
#box { border-radius: 12px; }

02 Common Properties

The building blocks of styling. Change colors, fonts, and backgrounds with these common rules.

color: #e5e7eb;
background-color: #111827;
font-family: 'Poppins', sans-serif;
line-height: 1.6;

03 The Box Model

Every element is a box. Control its size and spacing with content, padding, border, and margin.

.element {
  width: 200px;
  padding: 10px;
  border: 2px solid #60a5fa;
  margin: 15px;
}

04 Flexbox

A one-dimensional layout tool. Perfect for aligning items in a single row or column effortlessly.

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

05 CSS Grid

A two-dimensional layout system for creating powerful, complex page structures.

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 1rem;
}

06 Responsive Design

Make your site look great on any device using media queries. This is what makes a site truly modern.

@media (max-width: 768px) {
  .card {
    padding: 1rem;
  }
}
